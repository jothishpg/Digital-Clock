## 🕒 Digital Clock Web App

A simple and visually appealing digital clock built using HTML, CSS, and JavaScript. This project features a full-screen responsive layout with a modern glassmorphism design.

## ✨ Features
- Real-time digital clock display  
- Full-screen background image with fixed positioning  
- Centered layout using Flexbox  
- Glassmorphism effect using `backdrop-filter: blur()`  
- Large, bold monospace font for clear time visibility  
- Text shadow for enhanced readability  

## 🎨 UI Design
- Background image covers the entire screen and stays fixed during scrolling  
- Clock is centered both vertically and horizontally  
- Semi-transparent container with blur effect for a modern look  
- Clean and minimal design with focus on readability  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

## 📂 Project Structure
- `index.html` – Structure of the clock  
- `style.css` – Styling (background, layout, clock design)  
- `script.js` – Logic for updating time  

## 🚀 How It Works
The clock updates every second using JavaScript and displays the current system time in a large, centered format.
